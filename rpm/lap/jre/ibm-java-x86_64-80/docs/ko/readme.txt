Linux용 IBM SDK, Java Technology Edition 버전 8
=====================================================

이 README 파일은 새 README 파일에 별도로 명시하지 않는 한 버전 8,
릴리스 0, 수정사항 0, 수정팩 0(VRMF 8.0.0.0) 및 모든 후속 릴리스,
수정사항 및 수정팩에 적용됩니다.

IBM은 SDK와 추가 컨텐츠를 제공합니다.

제품 문서:
----------------------

이 릴리스를 지원하기 위한 사용자 안내서와 보안 안내서는
http://www.ibm.com/support/knowledgecenter/SSYKE2_8.0.0/welcome/welcome_javasdk_version.htm 사이트에서 온라인으로 제공됩니다.

사용자 안내서는 IBM developerWorks(https://www.ibm.com/developerworks/java/jdk/docs.htm)에서
다운로드할 수 있습니다.

최신 뉴스는 다음 IBM 지원 기술 노트에서 볼 수 있습니다.
