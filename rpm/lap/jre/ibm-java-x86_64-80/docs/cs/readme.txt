Sada IBM SDK pro systém Linux, Java Technology Edition, verze 8
=====================================================

Tento soubor README se vztahuje k verzi 8, vydání 1, modifikaci 0,
opravné sadě 0 (VRMF 8.0.0.0) a ke všem dalším vydáním, modifikacím
a opravným sadám, nebude-li v novém souboru README uvedeno jinak. 

Společnost IBM poskytuje se sadou SDK další obsah.
